# Design Anti-Patterns

Fable Mode should avoid common AI-generated UI habits unless they clearly serve the product. These patterns are not banned absolutely. Each one must earn its place through information architecture, interaction clarity, grouping, or brand expression.

## Common Tropes To Challenge

- Nested rounded cards.
- Gray borders around every region.
- Identical card grids.
- Eyebrow labels above every heading.
- Left-border accent panels.
- Arbitrary badges.
- Fake metrics.
- Generic dashboard tiles.
- Gradient backgrounds used as personality.
- Emoji as the visual system.
- Decorative blob illustrations.
- Excessive glassmorphism.
- Overused default font choices.
- Hierarchy made only through font weight.
- Center-aligned hero plus three-card layout as the default landing page.

## Before Using A Card

Ask:

- Is this truly a separate object?
- Does it need containment, or would spacing and typography be better?
- Does the border clarify structure or merely decorate?
- Can priority be expressed through layout, density, rhythm, or placement instead?

## Before Using A Badge

Ask:

- Does the label affect user judgment or action?
- Is it status, category, priority, or decoration?
- Would position, grouping, tone, or copy communicate the same idea better?

## Before Using A Grid

Ask:

- Is the content genuinely peer-level?
- Does the user need comparison across items?
- Should one item lead and the others support it?
- Does the grid collapse responsibly on smaller screens?

## Before Using A Gradient

Ask:

- Is the gradient part of a brand or material system?
- Does it improve hierarchy, mood, or state?
- Would a flat color, image, or typographic treatment be stronger?

## Better Defaults

Reach first for:

- Clear hierarchy.
- Purposeful spacing.
- Varied density.
- Strong alignment.
- Honest content.
- Real interaction states.
- Accessible contrast.
- Product-specific language.

If the design still looks like a generic template after those decisions, revisit the product purpose and information hierarchy before adding decoration.
